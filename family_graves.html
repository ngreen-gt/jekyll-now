I'd like to further expand these list to include basically any (great) grandparent, aunt, and uncle, and all my third (or fourth) cousins, on both sides of my family.

I will put an overall map, and then individual cemetery maps. I write more on the technical details and plans below the maps. Eventually I'd like a record for Greens, Hunsakers, Turners, and Lackeys.

Technical Implementation

It happened that I found a graphics tool that overlays data from a table over google maps. So I am using it to make a map of my family's grave sites.

I am using the example from http://hafen.github.io/rbokeh/#installation. The package used is called rbokeh. I've entered grave information into .csv file and use it with the R code from the example.
